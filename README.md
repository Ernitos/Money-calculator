# React Money-calculator
A simple and clean Money-calculator app built with React.


# Features

Convert between USD, EUR, RUB

Real-time calculation using React state

Clean and minimal UI

Uses useState for storing currency and input values




# Technologies Used

React

JavaScript (ES6+)

Vite

CSS

git clone https://github.com/Ernitos/todo-list.git

npm install
